About

I make the web
feel three dimensional.

I specialise in high-end interactive web experiences — from concept through to polished, performant code. Every project starts with a strong design direction and ends with an experience people remember.

3D Web & WebGL
Custom Three.js scenes, procedural geometry, real-time lighting and physics — built from scratch, not templates.
UI / UX Design
Clear design direction in every project — luxury, editorial, brutalist, playful. Intentional in every detail, every spacing decision, every colour choice.
E-Commerce & Retail
Product configurators, 3D showcases, category filters, cart systems. Built to convert, not just impress.
Performance & Mobile
All demos are mobile-tested, touch-enabled, and responsive. Three.js scenes resize cleanly. Interactions work on every device.

Three.js & WebGL
Procedural geometry, real-time shaders, dynamic lighting, particle systems. Every 3D model in this portfolio is built by hand from primitives.
🎨
CSS & Animation

JavaScript

📱
Responsive & Mobile
Every layout built mobile-first with fluid grids, adaptive typography, and Three.js scenes that resize correctly at any viewport width.
